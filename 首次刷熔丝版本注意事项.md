# 首次刷熔丝版本注意事项
##  <span style="color:red">检查是否已写入RPMB key</span>
**需要userdebug 版本，输入以下指令**
``` shell
adb root
adb shell
qseecom_sample_client -v smplap64 14 1
# 选择2选项
```
**如果设备输出的是已写入RPMB key的状态，就不要再刷efuse版本，否则会无法开机**

