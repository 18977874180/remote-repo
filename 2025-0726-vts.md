## 替换指令
``` bash
adb reboot bootloader
fastboot flashing unlock(需要保证设备处于unlock状态，如果设备处于unlock，忽略此步骤)
fastboot reboot fastboot

fastboot flash vendor_boot vendor_boot-debug.img
//指本地user版本包里的vendor_boot-debug.img，vts需要该步骤

fastboot flash system gsi
//gsi指google的system.img文件
fastboot -w
fastboot reboot
```
